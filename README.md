### Dependencies
* Node v14.18.2
* NPM v6.14.15
* React v17.0.2
* React router dom v6.3.0
* Typescript v4.2.4

### Installing
* Fork and clone this repository
* Run `npm install` in your terminal
* Run `npm start`
* (available to deploy your application to gh-pages)
